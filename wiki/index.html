欢迎使用ubuntu微型NAS服务器系统

如果你看到这个页面，说明系统的nginx网站成功运行.

NAS系统使用指南

1.初始账号和密码

root 1234
ubuntu 1234
系统默认北京时间
系统默认国内清华升级源
先插入U盘，后上电，系统才会默认自动挂载到/mnt/sda1，支持ntfs,ext4,fat32

2.个人网盘系统h5ai

浏览地址 http://192.168.xx.xx/files/

存档位置：/var/www/html/files/
更改到usb外接存储设备办法
例如：
mkdir -p /mnt/sda1/downloads
cd /var/www/html/files/
ln -s /mnt/sda1/downloads downloads

3.个人博客系统typecho

浏览地址 http://192.168.xx.xx/blog/

程序位置：/var/www/html/blog/
首次使用直接安装即可

4.默认配置nginx和php7.4-fpm,php7.4-cgi

配置参数请浏览 http://192.168.xx.xx/files/info.php
nginx配置文件位置/etc/nginx/sites-available/default

5.优秀的BT下载利器transmission和web管理页面

web管理页面地址 http://192.168.xx.xx:9091
下载文档位置：/var/lib/transmission-daemon/Downloads
帐号和密码都是transmission
更改设置方法：
service transmission-daemon stop
vi /etc/transmission-daemon/settings.json
修改完毕后service transmission-daemon start

5.1内置下载利器aria2和管理页面

web管理页面地址 http://192.168.xx.xx/ariang
下载文档位置：/root
配置文档位置: /usr/local/aria2/aria2.conf

6.修改静态ip方法：

vi /etc/network/interfaces.d/eth0
内容为
auto eth0
iface eth0 inet static
address 192.168.1.10
network 192.168.1.0
netmask 255.255.255.0
broadcast 192.168.1.255
gateway 192.168.1.1
dns-nameservers 192.168.1.1

7.内置了网页终端登陆管理后台ttyd

登陆地址 http://192.168.xx.xx:7681
让你更快捷的管理后台,
初始帐号root
密码1234
密码修改输入：passwd root根据提示修改

8.内置了阿里域名的自动ddns解析程序

使用方法：首先安装apt install nodejs
修改 vi /home/ubuntu/client-mode/config.json
填写你的阿里keyid和keysecret,填写你需要解析的域名
{
"AccessKeyId": "xxxx ",
"AccessKeySecret": "xxxx ",
"hostnames": ["@.ecoo.top", "www.ecoo.top"]
}
然后添加计划任务(每15分钟检查一次外网ip变化才自动更新)
crontab -e
*/15 * * * * /home/ubuntu/client-mode/client.sh

8.1内置了花生壳域名no-ip.com域名的自动ddns解析程序

使用方法：
花生壳修改 vi /home/ubuntu/client-mode/ddns_oray.sh
或noip修改 vi /home/ubuntu/client-mode/ddns_noip.sh
填写你的ddns帐号和密码,填写你需要解析的ddns域名
user=
passwd=
hostname=
然后添加计划任务(每20分钟检查一次外网ip变化才自动更新)
crontab -e
*/20 * * * * /home/ubuntu/client-mode/ddns_oray.sh
或者
*/20 * * * * /home/ubuntu/client-mode/ddns_noip.sh

9.内置激活微软产品的vlmcsd服务器

帮助文档请阅读： http://192.168.xx.xx/files/kms.html
助你永久免费激活微软产品.

10.安装和使用网络共享samba系统指南

安装方法apt install samba
修改配置vi /etc/samba/smb.conf 添加例如：
[downloads]
path = /mnt/sda1/downloads
read only = no
guest ok = yes
create mask = 0777
directory mask = 0777
browseable = yes

11.内置内网穿透frpc客户端

已内置了frpc客户端
使用方法：
vi /etc/frp/frpc.ini
例如：
将[xxxx-ssh]改为[hi3798mv1234-ssh]
将remote_port = 12999改为remote_port = 13998
最后执行systemctl enable frpc && reboot
这样，你就可以随时随地远程ssh -p 13998 root@frp.104300.xyz进入后台终端了。

12.安装和使用局域网音视频广播minidlna

apt install minidlna
修改vi /etc/minidlna.conf
将media_dir=/var/lib/minidlna修改为media_dir=/mnt/sda1/minidlna
在U盘创建目录mkdir /mnt/sda1/minidlna赋予777文件夹权限
在该文件夹下放一个mp3，一张图片，一个mp4视频
你就可以局域网全网收到广播。在电脑，手机，电视等智能终端进行播放。

13.内置vsftpd服务器

已安装ftp服务程序，可以使用ftp上传下载软件进行系统文件管理。
初始用户名ubuntu密码与系统一致。

14.安装和使用docker

apt install docker.io

15.恢复备份设置，即刷机时的backup还原

一句命令搞定：# recoverbackup 然后静等4分钟左右，即可继续登陆。
